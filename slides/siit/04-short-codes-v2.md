# Short Codes
## ทางลัดสำหรับ AI

---

## ปัญหา

พิมพ์ยาว = เสียเวลา

> "สร้าง GitHub issue เก็บ context แล้ว compact conversation"

---

## Solution

พิมพ์แค่: **ccc**

AI รู้ว่าต้องทำอะไร = เร็วขึ้น 10x

---

## Core 3 Short Codes

| Code | Purpose |
|------|---------|
| **ccc** | Context capture |
| **nnn** | Create plan |
| **gogogo** | Execute plan |

---

## ccc - Context Capture

**"เก็บ context ไว้ก่อน"**

1. รวบรวม state ปัจจุบัน
2. สร้าง GitHub issue
3. Compact conversation

---

## nnn - Create Plan

**"วางแผน"**

1. Check context (auto-ccc if needed)
2. Analyze issue/task
3. Create plan issue with steps

---

## gogogo - Execute

**"ลงมือทำ!"**

1. หา plan issue ล่าสุด
2. Execute step-by-step
3. Commit & push

---

## The Flow

```
ccc → nnn → gogogo

เก็บ context → วางแผน → ลงมือทำ
```

---

## สร้าง Short Code ของคุณเอง


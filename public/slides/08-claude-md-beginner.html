Tips

1. Start small เริ่มเล็กๆ — Just name + 2-3 rules

2. Add gradually เพิ่มทีละนิด — Found a problem? Tell AI to add rule

3. Be specific เจาะจง — "Don't read node_modules" > "Be careful"

4. Test it — New session, does AI follow?

4 How to Test

Ask a rule — "What time zone do you use?"

Check — AI should answer from CLAUDE.md

✓ Follows your rules = Working!
✗ Generic answer = Check file location

/trace - Parallel Agents

# /trace [project-name]

Launch 5 subagents simultaneously:

1. Current Repo Files  ‚Üí grep/find
2. Git History         ‚Üí git log --grep
3. GitHub Issues       ‚Üí gh issue list
4. Other Repos         ‚Üí find ~/Code
5. Retrospectives      ‚Üí grep œà/memory/

5 agents = 5x faster search
